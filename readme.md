# 🐱 Keystroke Cat Lifter

キーストロークの数をエネルギーに換算し、
「猫を何cm（m）持ち上げられるか」を計算するCLIツールです。

無駄に壮大で、ちょっとだけ達成感があります。

---

## 🔧 特徴

* Pythonのみで動作（依存ライブラリなし）
* importすら不要なシンプル設計
* 入力はキーストローク数だけ
* 子猫・猫・デブ猫を同時に計算
* なぜか猫を持ち上げる

---

## ▶️ 使い方

```bash
python main.py
```

実行後、キーストローク数を入力してください。

---

## ⌨️ キーストローク数の測り方

キーストローク数は自分で数えるのは大変なので、
以下のツールを使うのがおすすめです。

* **[WhatPulse](https://whatpulse.org/)**

  * キーボード入力数やマウスクリック数を記録してくれるアプリ
  * バックグラウンドで動作するので楽

---

## ⚙️ 計算について

* 1キーストローク = 0.02 J（仮定）
* 重力加速度 = 9.8 m/s²
* 高さは `h = E / (m × g)` で計算

---

## 🐱 対象の猫

| 種類  | 重さ  |
| --- | --- |
| 子猫  | 2kg |
| 猫   | 4kg |
| デブ猫 | 7kg |

---

## 🎯 最終目標

**デブ猫（7kg）を5m持ち上げること**

これはかなりのキーストロークを必要とします。
現実は厳しい。

---

## 💬 コンセプト

同じ努力（キーストローク）でも、
対象（猫の重さ）によって結果は大きく変わります。

* 子猫 → すぐ持ち上がる（成功体験）
* 猫 → そこそこ現実的
* デブ猫 → ほとんど動かない（壁）
